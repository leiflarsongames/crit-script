Crit-Script is a work-in-progress node-based scripting language intended for game masters. It is currently quite early 
in the development process.

Use `try_crit_script` to inspect the nodes available.
Note: Graphs are currently unavailable through `try_crit_script`.

See `test_core_library` and `test_make` to see the current state of Crit-Script, and see how graphs are constructed in 
Python directly.
